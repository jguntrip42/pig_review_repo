# Pig Optimal Play Reproduction

Joseph Guntrip, Kyle Russell, Tina Sardashti, Jay Styles

This project aims to reproduce the methodology and results from:
"Optimal Play of the Dice Game Pig" (Neller & Presser, 2004)

## Overview
We have implemented the value iteration approach to the game of Pig in order to compute the optimal strategy. In addition to the simplified game of Piglet.
